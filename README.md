# student-performance-analysis
Data analysis project using Google Sheets and Kaggle

## Objective
This project analyses student performance and explores how factors such as gender, age, and internet access are related to final results.

## Dataset
The dataset contains over 5,000 student records, including variables such as age, gender, study hours, attendance, and final performance.

## Data Cleaning
- Removed empty rows  
- Checked for missing values  
- Standardised categorical variables  
- Created age groups for analysis  

## Analysis
- Gender vs Study Hours  
- Age vs Performance  
- Internet Access vs Performance  

## Results
- Minimal difference in study hours between genders  
- Slight variation in performance across age groups  
- Internet access does not significantly impact performance  

## Conclusion
No single factor strongly influences performance. Other variables may play a more significant role.
